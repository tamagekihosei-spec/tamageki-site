# 多摩劇ホームページ

法政大学公認演劇サークル「多摩劇」のホームページ用ファイルです。

## ファイル構成

- `index.html`：サイト本体
- `style.css`：デザイン
- `script.js`：劇場演出、ページ切り替え、公演アーカイブ表示
- `assets/images/posters/`：公演ポスター画像
- `assets/images/gallery/`：公演写真
- `assets/flyers/`：フライヤーPDFなど

## GitHub Pagesで公開する方法

1. GitHubで新しいリポジトリを作成する
2. このフォルダ内のファイルをアップロードする
3. リポジトリの Settings → Pages を開く
4. Branch を `main`、Folder を `/root` に設定する
5. 数分待つと公開URLが発行される

## 公演アーカイブの更新方法

`script.js` の `performances` 配列に公演データを追加してください。

ポスター画像は `assets/images/posters/` に入れます。
公演写真は `assets/images/gallery/` に入れます。
